# TVSL AI-Powered Workforce Platform

A full-stack Next.js application for **Tech Valley Solutions Ltd.** (tvsl-bd.com) — an AI-powered employee and recruitment management system.

---

## 🚀 Getting Started

### 1. Install dependencies

```bash
npm install
```

### 2. Set up environment variables

Create a `.env.local` file in the project root:

```env
# Supabase (from https://supabase.com → your project → Settings → API)
NEXT_PUBLIC_SUPABASE_URL=https://your-project.supabase.co
NEXT_PUBLIC_SUPABASE_ANON_KEY=your-anon-key

# Gemini AI (from https://aistudio.google.com/app/apikey)
GEMINI_API_KEY=your-gemini-api-key
```

> **Note:** The app works without these keys — AI features will gracefully fall back to mock data. Only Gemini AI (chatbot, exam generation, CV scanning) needs `GEMINI_API_KEY`.

### 3. Run the development server

```bash
npm run dev
```

Open **http://localhost:3000** in your browser.

## ✨ Features

### Public Side
- Company landing page with services, stats, team info
- About page (CEO, values, contact)
- Job listings with filtering
- 3-step job application form (personal info → education → CV upload)
- **AI-generated exam** — 20 questions auto-generated by Gemini for the specific job
- 45-minute timed exam with MCQ + short answer questions
- Floating **AI chatbot** powered by Gemini

### Employee Side
- Dashboard with tasks, KPIs, quick actions
- Editable profile page
- Team viewer — all 5 teams with member cards
- Real-time-style team chat (3 channels)
- Notice board with priority filtering

### Admin Side
- Analytics dashboard (recruitment pipeline, activity feed)
- Employee management table (search, filter, detail panel)
- Job post manager (create, edit, delete)
- Applications manager (status updates, CV download, filtering)
- **AI exam evaluator** — send answers to Gemini, get scores + feedback
- **AI CV scanner** — paste/upload a CV, select a job, get match score + recommendation
